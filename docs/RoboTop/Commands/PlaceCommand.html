Overview

An instruction to place Robot at a specific (x, y) and orientation.

Constant Summary collapse

COMMAND_FORMAT =

Regexp with named captures that correlate with each Command type. To provide API consistency, unrecognised command patterns are named :null. See NullCommand and RoboTop::Commands::Parser#sanitized_command for more on how null matches are handled.

/\A(?:
  PLACE\s(?<x>\d+),(?<y>\d+),(?<f>#{Direction::ALL.to_a.join("|")})
)\Z/x.freeze

def initialize(instruction)
  super
  match_data = instruction.match(COMMAND_FORMAT)
  @x = match_data[:x].to_i
  @y = match_data[:y].to_i
  @f = match_data[:f]
  @orientation = Direction.named(@f)
end
